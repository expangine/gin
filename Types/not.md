# Not

**ID**: `not`  
**System Type**: `one of the defined types`  
**Sub Types**: `none`  
**Child Types**: `none`  

A value that can be anything BUT a given set of types.

### Type Options

- `[i]`: One of the inner types not allowed.

### Data Format

```
Data that's NOT of one of the inner types.
```

### Type Format

```
['not', [Type1, Type2, ...]]
```
