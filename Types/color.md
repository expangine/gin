# Color

**ID**: `color`  
**System Type**: `object`  
**Sub Types**: `r: num, g: num, b: num, a?: num`    
**Child Types**: *none*  

A color in RGB space, optionally with an alpha.

### Examples

- `[255, 255, 127, 0.5]`

### Type Options

- `alpha`: If alpha is part of the color.

### Data Format

```
['rgba', [r, g, b, a?]]
```

### Type Format

```
['rgba', { alpha?: boolean }?]
```
